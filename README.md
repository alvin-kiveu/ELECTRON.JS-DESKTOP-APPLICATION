How to set up

1. open your terminal/cmd

2. git clone https://github.com:alvin-kiveu/ELECTRON.JS-DESKTOP-APPLICATION.git

3. cd ELECTRON.JS-DESKTOP-APPLICATION

3. npm install

4. launch the app with npx electron .
